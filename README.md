Pixalara Recruitment Application
A recruitment management app built with React and Supabase for internal use.

Features
- Role-based dashboards (Admin, HR, Employee, Intern)
- Candidate management system
- Interview scheduling and evaluation
- Real-time candidate status tracking
- Audit logs for all activities
- User authentication and authorization

Tech Stack
- Frontend: React.js
- Backend: Supabase
- Styling: CSS
- Authentication: Supabase Auth

Setup Instructions
Prerequisites
- Node.js (v14 or higher)
- Supabase account

Installation
1. Clone the repository
2. Navigate to the frontend folder
3. Install dependencies: `npm install`
4. Create `.env` file with Supabase credentials
5. Run the application: `npm start`

Environment Variables
Create a `.env` file with:
- `REACT_APP_SUPABASE_URL`
- `REACT_APP_SUPABASE_ANON_KEY`

Project Structure
- `/src/components` - Reusable UI components
- `/src/pages` - Application pages
- `/src/context` - Authentication context
- `/backend/sql` - Database schema
